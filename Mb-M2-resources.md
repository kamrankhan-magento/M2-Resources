# DEVELOPER NOTES / QUICK REFERENCE

### Optimizing Magento 2 Dev Environment

```
• mode = developer
• cache on
• no DI compile
• opcode on (validate_timestamps on opcache.ini)
• PHP 7+
• xdebug off (use a toggle bash script)
```
